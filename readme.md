# SteamScraper Core-API
This is a core user registration and authentication api for a project called SteamScraper.

## Author:
Irakli Guraspashvili

## Stack:
- Server - Node.js, Express.js.
- Database - MongoDB.
- Frontend - Angular.
- API Gateway - Kong.
- Deployment - Docker.

## Architecture:
- Micro services.
- REST API.
